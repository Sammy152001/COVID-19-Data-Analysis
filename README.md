# COVID-19 Data Analysis and Prediction

## Description
This project analyzes global COVID-19 data using the Johns Hopkins dataset and applies Machine Learning techniques to understand trends and predict future case growth.

## Tools & Technologies
- Python (pandas, numpy, matplotlib)
- Machine Learning (scikit-learn)

## Dataset
Dataset is available in the `data/` folder.

## Objective
To analyze COVID-19 trends and build a Machine Learning model to predict future cases.

## Steps Performed
- Data cleaning and preprocessing  
- Time-series analysis  
- Feature engineering  
- Model building  
- Model evaluation  

## Key Insights
- Rapid increase in cases during early pandemic phases  
- Multiple waves observed globally  
- Growth trends identified from historical data  

## Machine Learning
- Model Used: Linear Regression / Time Series Model  
- Built predictive model to forecast COVID-19 cases  
- Compared actual vs predicted values  

## Final Result
- Successfully analyzed COVID-19 dataset  
- Built prediction model for case trends  
- Visualized future growth patterns  

## Conclusion
This project demonstrates how Machine Learning can be used for forecasting real-world events like pandemics.
