# COVID-19 Data Analysis

## Description
This project analyzes global COVID-19 data using the Johns Hopkins dataset to understand the spread and impact of the pandemic. It focuses on identifying trends, growth patterns, and recovery rates across different regions.

## Tools Used
- Python (pandas,numpy, matplotlib)

## Dataset
Dataset is available in the `data/` folder.

## Objective
To analyze COVID-19 trends and understand how the pandemic evolved globally.

## Steps Performed
- Data cleaning and transformation  
- Time-series analysis  
- Data visualization  

## Key Insights
- Rapid increase in cases during early pandemic phases  
- Multiple waves observed in different countries  
- Recovery rates varied across regions  

## Final Result
- Successfully analyzed COVID-19 dataset  
- Identified key trends and patterns  
- Visualized global spread of the virus  

## Conclusion
This project provides insights into the pandemic and helps in understanding how different countries were affected over time.
